# House Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts house prices using the Linear Regression algorithm. The model estimates the price based on features such as area, number of bedrooms, bathrooms, and house age.

## 🚀 Features
- Data preprocessing using Pandas
- Train-test data splitting
- Linear Regression model training
- House price prediction
- Model evaluation using MAE and R² Score

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 📂 Project Files
- house_prediction.py
- house_price.csv
- README.md

## ▶️ How to Run
1. Install the required libraries:
   ```
   pip install pandas scikit-learn matplotlib
   ```
2. Run the project:
   ```
   python house_prediction.py
   ```
